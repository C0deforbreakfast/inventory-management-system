# Inventory Management System

A desktop-based inventory management application developed using Python and Tkinter. This project was created as part of a Database Systems course and provides a graphical user interface for managing inventory records, tracking stock levels, and performing inventory-related operations.

## Features

* User-friendly graphical interface built with Tkinter
* Add new products to the inventory
* Remove products from the inventory
* Search for products
* Manage storage quantities
* Update inventory records
* Persistent data storage using CSV files
* Modular application design

## Technologies Used

* Python
* Tkinter
* CSV File Handling
* Object-Oriented Programming (OOP)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/C0deforbreakfast/inventory-management-system.git
```

2. Navigate to the project directory:

```bash
cd inventory-management-system
```

3. Run the application:

```bash
python main.py
```

## Academic Context

This project was developed for a Database Systems course to demonstrate inventory management concepts, data organization, file-based persistence, and desktop application development using Python.
